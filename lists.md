# Lists Examples

Blank List

```python
cats = []
```

List with one item

```python
cats = ["garfield"]
```

List with three items

```python
cats = ["garfield", "snowball", "fluffy"]
```

Print out first item from the list (at index 0) 

```python
cats = ["garfield", "snowball", "fluffy"]
print(cats[0])
#      ^   ^
#list name |
#          index number
#
# Result:
# garfield
```
